Powerful tool to tweak your existing loops. Set posts per page, filter posts, order them, hide them, apply Conditional Logic, etc.

== Description ==

Posts Per Page customizer allows you to carefully declare how many posts per page you want to show, and where you want to show them using
our Conditional Logic. It also allows you to filter posts, order them by comments count, date, random, and other possibilities.

== Installation ==


1. Upload `posts-per-page-customizer.zip` file in your WordPress Admin -> Plugin -> Add New
2. Activate the plugin through the 'Plugins' menu in WordPress
